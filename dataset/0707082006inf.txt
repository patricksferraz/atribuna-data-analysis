Câmera de 3.2 megapixels
<b> Novos modelos de celular
fotografam em alta qualidade,
tocam 23 horas de música e
anexam fotos no blog do usuário </b><br>
 <br>
Quem quer dar um celular novo para o papai e impressioná-lo de verdade com o presente, uma ótima dica é escolher entre um dos dois novos modelos de celulares da Sony Ericsson: o K790i, com câmera de 3.2 megapixesls, e o W300i, com tocador de MP3.
O K790i, por exemplo, é o primeiro celular que usa a marca CyberShot, vem com uma câmera 3.2 megapixels embutida com  flash de Xenon, que tem alcance bem maior que os flashes com LED que costumam acompanhar os celulares. 
Outro recurso interessante é o BestPic. Com ele acionado, a câmera é capaz de capturar nove quadros em um segundo. É um recurso interessante para quem quer fotografar cenas em movimento. Depois de clicadas, as imagens aparecem separadas na tela. O usuário pode escolher e salvar todas ou as que preferir. 
Para evitar fotos tremidas, há ainda um software estabilizador embutido, que funciona tanto para fotos como para filmes. O armazenamento é feito nos 64 MB do cartão modelo Memory Stick Micro (M2), que é bem menor que o Memory Stick tradicional. Mas, para permitir o uso do minúsculo cartão nos leitores convencionais, o celular vem acompanhado de um adaptador. 
Para usuários de blogs, o K790i vem acompanhado de um atalho para o Blogger. É só tirar a foto e escolher a opção “Enviar para o Blogger”. O celular faz tudo sozinho. Caso o usuário não tenha uma conta no serviço, ela é aberta automaticamente. 
Outro recurso é a possibilidade de visualizar os mapas do Google Earth. As imagens podem ser vistas sem problema na tela de 1,8 polegadas do aparelho.
Outra superdica de presente, feito sob medida para o papai que gosta de música, é o aparelho W300i, também da Sony Ericsson.
O aparelho suporta arquivos de música em MP3 ou AAC, tem rádio FM e câmera VGA. Para suportar horas ligado vem com uma bateria de fôlego. Segundo a Sony Ericsson, com o player ligado e o aparelho em stand by é possível ouvir 23 horas de música. Para incluir as músicas no aparelho é preciso usar o software Disc2Phone. 
Junto com o aparelho vem um kit que inclui o cartão de memória Memory Stick Micro (M2), um cabo para conexão com o PC e um fone de ouvido.
Os dois celulares têm tecnologia GSM. O W300i custa R$ 799,00 e o K790i chega às lojas por R$ 1.799,00.
