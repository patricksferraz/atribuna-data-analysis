O uso de drives externos tem se tornado cada vez maior, tanto por empresas como profissionais. Além de armazenar e transportar arquivos de maneira segura, os HDDs (Hard Disk Drives), ou HDs externos, como são mais conhecidos, se tornaram alternativas econômicas para usuários de desktops e notebooks.
Essa tecnologia possibita a ampliação da capacidade de armazenamento sem a necessidade de fazer upgrades ou trocar de equipamento toda vez que o disco rígido fica lotado.
A Clone está lançando dois modelos de gavetas externas para HDs de 2,5 polegadas para notebooks, e de 3,5 polegadas para discos rígidos. 
Ideais para o transporte e armazenamento de dados com rapidez e segurança, os produtos atingem uma taxa de transferência de dados de até 480 Mbps através da conexão com a porta USB 2.0. 
De fácil instalação e transporte, as gavetas foram confeccionadas em material resistente para evitar qualquer tipo de batida ou arranhão no HD.
Além disso, o modelo de 3,5 polegadas acompanha um suporte plástico para acomodá-lo mais facilmente na mesa. Os lançamentos da Clone figuram entre as soluções para quem trabalha com grandes volumes de dados e precisa transportá-los com eficiência, praticidade e segurança.
As gavetas para HD de 2,5 polegadas custam cerca de R$ 79,00. Já o valor estimado do modelo para discos rígidos é de R$ 136,00.
