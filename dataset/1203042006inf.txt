Vista só em 2007
<b> Microsoft atrasa o lançamento
de seu novo sistema
operacional, que deve acontecer
só no começo do próximo ano </b><br>
 <br>
O Windows Vista, próxima versão do sistema operacional da Microsoft que vai suceder o XP, será lançado para os consumidores no começo de 2007 e não mais no final do segundo semestre de 2006, segundo informação do co-presidente da divisão de plataformas e serviços da empresa, Jim Allchin.
O atraso do Windows Vista foi causado porque a Microsoft precisa de mais tempo para melhorar os recursos de segurança e outras funções. “Só estamos tentando ser responsáveis e francos. Estamos atrasados algumas semanas em termos de qualidade. Não é um novo trabalho que estamos acrescentando. Continuamos a trabalhar para fazer o Vista o sistema mais seguro que já se desenvolveu, afirmou Allchin.
A companhia adiantou que vai lançar o sistema operacional em novembro, como o planejado, mas só para grandes empresas. Os consumidores terão de esperar até janeiro.
Essa é uma péssima notícia para os fabricantes de computadores, que contavam com o sistema operacional para alavancar as vendas do Natal, em dezembro.
Quem pode se favorecer com o atraso do Windows Vista é a Apple, que promete novidades para esse período e pode ganhar a preferência dos consumidores norte-americanos.
O Windows Vista é o primeiro sistema operacional voltado para os consumidores desde que a Microsoft lançou o XP, há quase cinco anos, no final de 2001.
A Microsoft disse que o atraso do Vista não vai afetar o lançamento do Internet Explorer 7 para o Windows XP no segundo semestre deste ano. <br><br><hr>
