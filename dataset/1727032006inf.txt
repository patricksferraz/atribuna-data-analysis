Moderninhos e poderosos
<b> Novo sistema operacional
da Microsoft é sinônimo
de produtividade para
desenvolvedores  </b><br>
 <br>
Eles vieram para ficar e ganharam novos sistemas operacionais para aprimorar a capacidade de seus dispositivos. Os handelds são os queridinhos de quem gosta de praticidade aliada a tecnologia. E se no passado, um dos maiores tormentos para os donos desses “bichinhos” móveis era a perda de dados e arquivos toda vez que a bateria dos equipamentos se esgotasse, o problema já tem solução. A Microsoft acaba de lançar uma nova versão do seu sistema operacional, chamada de Windows Mobile 5 (WM 5.0), inaugurando a era dos pockets PCs com memória persistente.
A novidade, que chegou para bombar o pequenino, é a adição de memória Flash no armazenamento dos dados e a utilização de memórias RAM e ROM. Agora a arquitetura dos handhelds segue um esquema semelhante adotado pelos desktops. Ou seja, a RAM funciona exclusivamente para rodar programas, não servindo como área de instalação de aplicativos. E, todo o armazenamento é feito na memória flash ROM, que não depende de energia para armazenar os dados. Traduzindo, o usuário não precisa mais se preocupar com a perda de dados quando a bateria do aparelho é zerada.
O sistema tem tudo para cair no gosto popular, pois foi projetado para facilitar ainda mais a vida dos usuários. O recurso de tela no modo paisagem (deitada) permitirá usuários a ter acesso a uma tela mais larga para leitura de planilhas e páginas web. Suporte para novos botões de acesso rápido, que podem ser usados nos dois modos de tela, irá permitir o uso do palmtop com menos acesso à caneta de escrita.
O Windows Mobile 5.0 suporta aparelhos com disco rígido e interface USB 2.0, permitindo aos usuários gerenciar uma quantidade de dados antes impensável em um dispositivo portátil. Com relação a função multimídia, o WM5 tem um novo e potente aplicativo para ver e editar vídeos e fotos, ele foi adicionado para acompanhar a crescente demanda por visualizadores multimídia. E pode reduzir, ampliar, fazer slides show e etc. 
Quando o assunto é música, os usuários podem ficar tranquilos porque agora podem desfrutar de um grande número de músicas digitais protegidas, gravar programas de televisão que podem ser facilmente sincronizados entre o PC e o palmtop, e ainda fazer downloads de sites de músicas. Desenvolvedores podem adicionar Plug-Ins como o gerenciamento de direitos digitais (DRM), para permitir comércio de conteúdo protegido. 
Segurança é um dos pontos fortes do WM 5. Apesar da versão anterior já ter sido considerada um exemplo de segurança, a versão 5.0 é ainda mais seguro. Este sistema foi aprovado pelo departamento interno da Microsoft, recentemente criado para checar o nível de segurança de seus aplicativos. Não há brecha para hackers. 
Outra vantagem do Windows Mobile 5 é que o sistema traz uma versão melhorada do Office, com funcionalidades integradas. A mudança traz facilidades para executivos, profissionais e até estudantes, no momento da utilização do palmtop em substituição ao desktop. O PowerPoint Mobile foi incluído no pacote. Com ele os usuários vão poder visualizar apresentações na telinha sem recorrer a aplicativos de terceiros. Na prática, o usuário poderá conferir uma apresentação, antes de encaminhá-la por e-mail.
Para os desenvolvedores de sistemas a nova versão do sistema é sinônimo de produtividade no desenvolvimento, já que o Windows Mobile 5 dá possibilita um ambiente de desenvolvimento único, com conceitos e bibliotecas bem parecidas, para desenvolver sistemas tanto para smartphones quanto para palmtops. 
As novas tecnologias que vão interessar e muitos a estes profissionais são o aumento da flexibilidade da plataforma, que irá permitir aos desenvolvedores mais liberdade de interação com código fonte. Recursos de vídeos e fone conferência já estão embutidos neste novo sistema. 
Vários fabricantes já lançaram modelos de equipamentos usando o WM 5.0. Além disso, a HP e a DELL confirmaram que o WM 5.0 pode ser instalado em alguns de seus modelos já lançados no mercado. <br><br><hr>
