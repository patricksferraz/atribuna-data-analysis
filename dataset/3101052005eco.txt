DIREITO ELEITORAL

Questão 01

Segundo a nossa Constituição de 1998, NÃO é órgão da Justiça Eleitoral.
1- (	) O Tribunal Superior Eleitoral.
2- (	) As Juntas de conciliação Eleitorais.
3- (	) Os Juízes Eleitorais.
4- (	) Os Tribunais Regionais Eleitorais
5- (	) As Juntas Eleitorais.

Questão 02

Acerca da legislação eleitoral, a respeito dos recursos as decisões dos TREs, são recorríveis as decisões exceto:
1- (	) Forem proferidas contra disposição expressa desta Constituição ou de lei.
2- (	) Ocorre divergência na interpretação de lei entre dois ou mais tribunais eleitorais.
3- (	) Versarem sobre prescrição do prazo de candidatura.
4- (	) Anularem diplomas ou decretarem a perda de mandatos eletivos federais ou estaduais.
5- (	) Denegarem habeas-corpus, mandado de segurança, habeas data ou mandado de injunção.

Questão 03

Julgue os itens a seguir.
1- (	) Pode o eleitor possuir vários domicílios eleitorais, para facilitar o seu exercício de cidadão, pois assim mesmo que esteja longe de seu domicílio poderá voltar.
2- (	) Pedro Inácio Collor Cardoso, brasileiro nato, nunca foi processado, está quites com todas as obrigações eleitorais e militares. Considerando que este possui 35 anos, poderá concorrer a Presidência da República, pois ele possui todos os requisitos legais que a lei exige.
3- (	) Pode brasileiro naturalizado ser eleitor.
4- (	) Pode candidato oferecer transporte aos eleitores, para que estas não percam seu direito-dever de votar.
5- (	) Todo partido político tem direito a propaganda eleitoral gratuita.

Questão 04

Terá cancelado seu alistamento eleitor que deixar de:
1- (	) Votar em 03 (três) eleições interpoladas.
2- (	) Vier a residir em localidade diversa de onde se alistou.
3- (	) Votar em 03 (três) eleições intercaladas.
4- (	) Votar em 03 (três) eleições consecutivas.
5- (	) Cometer o crime de homicídio contra qualquer representante do poder legislativo seja federal, estadual ou municipal.

Questão 05

Segundo o Código Eleitoral a respeito de registro de candidatos análise as proposições.
1- (	) Nenhum registro de candidato será admitido fora do período de 6 (seis) meses antes da eleição.
2- (	) Admite-se registro de candidato para cargos diferentes, por mais de uma circunscrição ou para mais de um cargo na mesma circunscrição.
3- (	) Somente podem ocorrer às eleições candidatos registrados por partidos.
4- (	) Nas eleições realizadas pelo sistema proporcional o candidato deverá ser filiado ao partido, na circunscrição em que concorrer.
5- (	) Independente de filiação partidária o candidato poderá concorrer ao pleito eleitoral, pois não se pode impedir o acesso a cargo eletivo.

Questão 06

Ainda sobre registro eleitoral assinale a alternativa correta
1- (	) Serão registrados no Tribunal Superior Eleitoral os candidatos a presidente e vice-presidente da República.
2- (	) Serão registrados no Tribunal Superior Eleitoral os candidatos a deputado federal e senadores.
3- (	) Serão registrados nos Tribunais Regionais Eleitorais os candidatos a senador, deputado federal, governador e vice-governador e prefeitos.
4- (	) Serão registrados nas Juntas Eleitorais os candidatos a governador e vice-governador e prefeitos.
5- (	) Serão registrados nos Tribunais Regionais Eleitorais os candidatos a senador, deputado federal, vereadores e prefeitos.

Questão 07

Constitui-se crime eleitoral:
1- (	) Inscreve-se fraudulentamente eleitor.
2- (	) Perturbar ou impedir de qualquer forma o alistamento eleitoral.
3- (	) Dar, oferecer, prometer, solicitar ou receber, para si ou para outrem, dinheiro, dádiva, ou qualquer outra vantagem, para obter ou dar voto e para conseguir ou prometer abstenção, ainda que oferta não seja aceita.
4- (	) Majorar os preços de utilidades e serviços necessários a realizações de eleições, tais como transporte e alimentação de eleitores, impressão, publicidade e divulgação de matéria eleitoral.
5- (	) Todas as alternativas acima.

Questão 08

Assinale a assertiva correta.
1- (	) As infrações penais definidas neste Código são de ação pública condicionadas a representação do ofendido.
2- (	) Não pode o Ministério Público Eleitoral atuar em crimes eleitorais sem a devida autorização do Juiz competente.
3- (	) Qualquer cidadão que tiver conhecimentos de infração penal eleitoral deverá comunica-la ao juiz eleitoral da zona onde a mesma se verificou.
4- (	) Somente as alternativas A e C estão corretas.
5- (	) Somente a alternativa C está correta.

Questão 09

Acerca da legislação em vigor assinale a assertiva correta.
1- (	) Não poderá o eleitor mudar o domicílio eleitoral, pois este procedimento é de caráter definitivo.
2- (	) O eleitor poderá ter tantos domicílios eleitorais quanto julgar necessários ao seu direito de sufrágio.
3- (	) Somente é obrigatório o voto aos maiores de dezoito e menores de setenta anos.
4- (	) Qualquer cidadão é alistável e elegível, pois a constituição garante que todos são iguais perante a lei.
5- (	) Não pode ser candidato quem já cumpriu pena por crime eleitoral.

Questão 10

Julgue os itens a seguir.
1- (	)O Procurador Geral da República, não é necessariamente o Chefe do Ministério Público Eleitoral.
2- (	)O TSE compõe-se de onze ministros nomeados pelo Presidente da República e mediante aprovação do Senado.
3- (	)Admite-se que analfabetos alistem-se facultativamente.
4- (	)Pode concorrer a cargo de vereador pessoa com idade de dezoito anos.
5- (	)Pode advogado escolhido para exercer um cargo de ministro do TSE ser Corregedor, mas não Presidente do Tribunal.

DIREITO CONSTITUCIONAL

Questão 01

(Cespe/Fiscal/INSS/1998) A respeito dos direitos fundamentais da Constituição de 1998, julgue os itens seguintes.
1- (	) Sabendo que, segundo a Constituição, é livre o exercício de qualquer trabalho, ofício ou profissão, atendidas as qualificações profissionais que a lei estabelecer, é correto concluir que enquanto não sejam definidas por lei as qualificações necessárias para o desempenho de certa atividade profissional, ela não poderá ser exercida.
2- (	) Qualquer indivíduo, desde que brasileiro, é parte legítima para ajuizar ação popular que vise a anular ato lesivo ao patrimônio público.
3- (	) A Constituição não admite penas de caráter perpétuo ou de trabalhos forçados.
4- (	) Mesmo sabendo que a Constituição estabelece que ninguém será considerado culpado até o trânsito em julgado de sentença penal condenatória, não é correto afirmar que o indivíduo somente possa ser legitimamente preso depois de transitado em julgado a sentença condenatória.

Questão 02

(Cespe/ Analista judiciário STJ/1999) Acerca dos princípios, dos direitos e das garantias fundamentais previstos na Constituição da República, assinale a opção correta.
1- (	) A condenação de um Presidente da República por improbidade enseja a cassação dos seus direitos políticos.
2- (	) A moralidade pública sobrepõe-se às questões processuais. Assim é admissível, para efeitos penais, a prova obtida contra agente público por meio de votação, sem ordem judicial, do sigilo de suas comunicações telefônicas, desde que em assunto afeto à função pública exercida.
3- (	) A concessão de asilo político a governante estrangeiro não se coaduna com os princípios constitucionais que regem o Brasil nas suas relações internacionais, à medida que, interferindo em assuntos afetos a outro país, violam-se os ditames de respeito à independência e à autodeterminação dos povos.
4- (	) Considerando a rigidez das regras definidas no texto constitucional-editado em movimento histórico diverso do presentemente vivido -, não é possível a redução de salários dos trabalhadores como forma de reduzir demissões, nem mesmo por meio de convenção ou acordo coletivo.
5- (	) O trabalho insalubre em minas de carvão ou na operação de máquinas que possam causar mutilação só é permitido a partir dos dezoito anos de idade.

Questão 03

Julgue os itens.
1- (	) A proibição constitucional de cassação de direitos políticos impõe, como conseqüência, a impossibilidade de cassação de mandado eletivo.
2- (	) A criação de estados e de municípios depende de lei complementar, cujo projeto não pode ser oferecido sem que haja ocorrido a aprovação da emancipação pelas populações diretamente interessadas, residente dentro ou fora da área emancipada, em plebiscito.
3- (	) Rios que sejam bens da União implicam a propriedade federal também das ilhas de seu curso.
4- (	) Não pode haver propriedade municipal ou estadual em ilhas oceânicas.
5- (	) Particular não tem direito a participação pela exploração de jazidas minerais em terras de sua propriedade.

Questão 04

Julgue os itens.
1- (	) Homens e mulheres são detentores de igualdade absoluta na Constituição vigente.
2- (	) A Constituição aponta situações de tratamento desigual em relação ao sexo.
3- (	) É constitucional a existência das leis infraconstitucionais, complementares ou ordinárias, criando situações de tratamento desigual em razão do sexo ou regulamentando as situações constitucionais de desigualdade sexual.
4- (	) Uma lei elaborada anteriormente à Constituição vigente e que consagre situação de desigualdade sexual não amparada por essa Constituição, será inconstitucional.

Questão 05
 
(Analista judiciário/9ª região) Na hipótese de o Presidente da República não efetuar junto ao Congresso Nacional, dentro de 60 (sessenta) dias após a abertura da sessão legislativa anual, a apresentação de contas referentes ao exercício anterior, deve-se proceder à tomada de contas pelo(a):
1- (	) Congresso Nacional.
2- (	) Câmara dos Deputados.
3- (	) Senado Federal.
4- (	) Tribunal de Contas da União.
5- (	) Ministério Público Federal.

Questão 06

(Esaf/ Assist. jurídico/AGU/1999) Assinale a opção correta.
1- (	) Segundo entendimento pacífico do Supremo Tribunal Federal, qualquer alteração que afete os direitos fundamentais configura lesão expressa à cláusula pétrea.
2- (	) Segundo a jurisprudência do Supremo Tribunal Federal, não só as normas constantes do catálogo de direitos fundamentais, mas também outras normas consagradoras de direitos fundamentais constantes do Texto Constitucional podem estar gravadas com a cláusula de imutabilidade.
3- (	) Os direitos previstos em tratados internacionais firmados pelo Brasil somente poderão ser alterados mediante emenda constitucional.
4- (	) É vedada a alteração de disposições transitórias constantes do texto constitucional original.
5- (	) Segundo afirme jurisprudência do Supremo Tribunal Federal, é admissível a argüição de inconstitucionalidade de norma constitucional ordinária. 

Questão 07

(Cespe/ Del. Polícia Federal/1997) Com relação aos direitos políticos, julgue os itens abaixo.
1- (	) Os direitos políticos compreendem as faculdades de votar e de ser votado e nelas esgotam-se.
2- (	)O alistamento eleitoral é obrigatório para todos os brasileiros alfabetizados e maiores de dezoito anos.
3- (	)O sufrágio no Brasil possui índole capacitaria e censitária.
4- (	)No Brasil, quem tem capacidade eleitoral passiva tem, necessariamente, a ativa; todavia, nem todo eleitor é elegível.
5- (	)Inelegibilidade é diferente inalistabilidade, que, que por sua vez, é diferente de incompatibilidade.

Questão 08

(Procurador do RS / 1997) A competência denominada literalmente de concorrente pela Constituição de 1988.
1- (	) Cabe à União, aos Estados, ao Distrito Federal e aos Municípios.
2- (	) Exclui a competência da União de legislar normas específicas para sua própria esfera de poder.
3- (	) Confere, em qualquer caso, liberdade legislativa plena aos Estados.
4- (	) Pode colocar limites à atuação dos poderes municipais.
5- (	) Importa a revogação da lei estadual, na hipótese de superveniência de lei do Congresso Nacional que lhe for contrária.

Questão 09

Assinale as questões como certas ( C ) ou erradas ( E ).
1- (	) A lei vai fixa os subsídios dos Ministros do Supremo Tribunal Federal deve ter seu projeto iniciado, conjuntamente, pelos Presidentes da República, da Câmara dos Deputados, do Senado Federal e do próprio STF.
2- (	) O Congresso Nacional tem competência para firmar tratados e acordos internacionais em nome do Brasil, quando acarretarem compromissos gravosos ao patrimônio nacional.
3- (	) O Congresso tem competência para julgar as contas do Presidente da República e dos demais ordenadores de despesa da Administração direta federal.
4- (	) O Congresso tem competência para apreciar os atos de concessão ou renovação de concessão de emissoras de rádio e televisão, mas o ato de concessão, de competência privativa do Presidente da República, somente será aprovado por maioria de dois quintos, em votação necessariamente nominal.
5- (	) A alienação de terras públicas com área superior a 2.500 hectares depende, em qualquer caso, de aprovação do Congresso Nacional.

Questão 10

(Esaf/AFTN/1996) Julgue os itens.
1- (	) A sanção do Presidente da República a projeto de lei aprovado pelo Congresso Nacional é hábil para convalidar eventual vício de iniciativa.
2- (	) É admissível a rejeição pelo Congresso Nacional de veto total a um projeto de lei.
3- (	) A Constituição Federal assegura iniciativa popular também para as Emendas Constitucionais.
4- (	) Segundo orientação do Supremo Tribunal Federal, não podem ser objetos de medida provisória as matérias que não podem ser objeto de delegação.
5- (	) O Presidente da República pode retirar da apreciação do Congresso Nacional medida provisória recém-editada.

DIREITO ADMINISTRATIVO

Questão 01

Ato lesivo ao erário público poderá ser impugnado por qualquer cidadão mediante:
1- (	) Mandado de segurança.
2- (	) Mandado de injunção.
3- (	) Ação popular.
4- (	) “Hábeas data”.
5- (	) Mandado de segurança coletivo.

Questão 02

Incluem-se entre os bens dos Estados-membros:
1- (	) As ilhas lacustres, não pertencentes à União.
2- (	) As ilhas fluviais, pertencentes à União.
3- (	) As ilhas oceânicas, pertencentes à União.
4- (	) As terras devolutas, compreendidas entre as da União.
5- (	) Os terrenos de marinha.

Questão 03

Conforme a atual Constituição e a Emenda Constitucional n.19, a Administração Pública obedecerá aos seguintes princípios:
1- (	) Legalidade, impessoalidade, moralidade, publicidade e eficiência.
2- (	) “Nullum crimen, nulla poena sine lege”.
3- (	)  Abuso de poder.
4- (	) Excesso de poder.
5- (	) Anterioridade da lei.

Questão 04

Conforme a Constituição de 1998, a Administração direta, indireta ou fundacional existe:
1- (	) No Poder Executivo apenas.
2- (	) No Poder Judiciário e Legislativo.
3- (	) Em qualquer dos Poderes, das várias esferas.
4- (	) No Poder Executivo e Judiciário.
5- (	) No Poder Legislativo e Executivo.

Questão 05

A investidura, em cargo ou emprego público, depende de:
1- (	) Tempo de serviço, público, superior a 5 anos.
2- (	) Não haver sido, o nomeado, demitido em processo anterior.
3- (	) Folha de serviços irrepreensível.
4- (	) Conduta nunca posta em dúvida, por órgão público.
5- (	) Aprovação prévia, em concurso público, de provas ou de provas e títulos.
